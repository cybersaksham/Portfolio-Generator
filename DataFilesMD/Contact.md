# Contact.js File

- This file contains contact information of the user.
- Those details are as follows.

```txt
- Twitter Link
- Instagram Link
- Github Link
- Linkedin Link
- Phone number
- Email
- Website URL
```

## Code Structure

```js
const contact_links = {
  twitter: twitterLink,
  instagram: instagramLink,
  github: githubLink,
  linkedin: linkedinLink,
  phone: phone,
  email: email,
  website: website,
};

export default contact_links;
```
