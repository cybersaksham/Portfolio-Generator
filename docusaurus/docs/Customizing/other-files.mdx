## Resume.pdf {#resume}

- It is pdf file of your resume. The pdf file is not necessary to generate website.
- `portfolio-generator` will ask for confirmation whether you want to upload `Resume.pdf`
- You can view default bg.webp [here](https://github.com/cybersaksham/Portfolio-Generator/blob/master/Required%20Code/public/Resume.pdf).
